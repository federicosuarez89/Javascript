# javascript
Repositorio con mi desarrollo del lenguaje javascript. Diferencia entre var y let para declarar variables. Scope global o local de las variables. Uso de la palabra cons para datos primitivos y valores compuestos. Strings y sus métodos. Template Strings (interpolación de cadenas). Tipos de datos Numbers y sus metodos. Tipos de datos primitivos boolean, null, undefined y NaN. Tipo de dato compuesto, funciones declaradas y expresadas con su comportamiento debido al hoisting de Javascript. Tipo de dato compuesto, arreglos y sus métodos.
